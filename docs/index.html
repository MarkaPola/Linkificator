Linkificator, a Firefox Add-on to convert text links into clickable links.


This add-on parse text parts of HTML pages to match hypertext patterns not correctly encoded (i.e. not part of an anchor) and add an anchor to enable a standard mouse click to access the target as specified by the hypertext.

For testing purposes, I have compiled a fairly exhaustive list of test cases. Linkificator should recognize each of these, unless otherwise marked.
