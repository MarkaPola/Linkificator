History of changes

Version 1.1.0

New features

  • Possiblity to re-linkify the current page.
    This action can be launched by a right click on the Linkificator widget or through the hot key control-shift-u (customizable)
  • Toggle the state of Linkificator is now attached to the middle click on the widget
  • Left click on the widget now display a context menu enabling an easy access to the most common operations as well as the options page

Version 1.0.3

Enhancements and bug fixes

  • Enable custom rules edition on Mac OS
  • Enhance statistics handling
  • Fix erroneous inline elements processing

Version 1.0.2

Enhancements and bug fixes

  • Enhance URLs parsing
  • Reduce resources consumption and potential UI freezes

Version 1.0.1

Enhancements and bug fixes

  • Tag <span> is now handled has an inline element
  • Enhance localization messages

Version 1.0.0

New features

  • Inline elements, like <big> or <strong>, are now recognized and correctly handled during URLs parsing
  • Existing pages are now automatically parsed when add-on is installed or re-enabled. It is no longer required to manually reload the page

Enhancements and bug fixes

  • Multi-frames documents are now correctly handled in all cases
  • plain text documents are now recognized and parsed
